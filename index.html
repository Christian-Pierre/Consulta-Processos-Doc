Sistema de Consulta de Processos Judiciais

API REST em Django para cadastro, consulta e monitoramento de processos judiciais nos tribunais estaduais brasileiros

Visão Geral do Projeto

Este projeto apresenta uma solução completa para cadastro, consulta e monitoramento de processos judiciais nos tribunais estaduais brasileiros, utilizando Django (Python) como framework principal e seguindo o padrão REST para a API.

Objetivos

O sistema permite aos usuários:

  • Cadastrar processos para monitoramento
  • Consultar processos por número, CPF ou CNPJ
  • Acompanhar atualizações e movimentações
  • Receber notificações sobre mudanças de status

Características Principais

Consulta Inteligente

Busca processos por número, CPF ou CNPJ em todos os tribunais estaduais brasileiros.

Atualização Automática

Job periódico que mantém os dados atualizados com priorização inteligente.

Sistema de Notificações

Alertas personalizáveis sobre mudanças de status e novas movimentações.

Segurança e Escalabilidade

Arquitetura robusta com autenticação JWT e design escalável.

Tecnologias Utilizadas

  • Backend: Django 4.2+, Django REST Framework
  • Processamento Assíncrono: Celery, RabbitMQ
  • Banco de Dados: PostgreSQL
  • Cache: Redis
  • Integração: API Datajud/CNJ, APIs de tribunais estaduais
